what is jQuery

It is a code library that somebody wrote ,you can incorporate into your own project.

how to incorporate jQuery into website

  1. 導入css (head 部分)

  2. 導入js (body close tag上方一行)

  3. 到jQuery網頁複製CDN(使用google CDN),在原先的javascript code上方一行貼上(一定要放在原來的js code上面一行,否則會無法生效,因為程式碼一行行跑下來必須先到js頁面,這樣js頁面使用jQuery語法的程式碼才會生效)

  4. 如果jQuery放置在head部分無法執行,可以使用下方程式碼

how minification works to reduce file size

https://www.minifier.com/ 能夠把jscode 以及css code簡化的網站

把容易閱讀的程式碼簡化,可以減少不少容量,因為要使瀏覽器看得懂程式碼其實不需要那麼多的空格,以及排版,因此簡化過後檔案容量會縮小
