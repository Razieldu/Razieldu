Introduction to the Document Object Model

Dominating the DOM to add functionality to HTML elements

Basically catalogs the web page into individual objects that we can select and manipulate,when you load up your page,what it does is that it turns each of these elements and their associated data into a tree structure with a whole bunch of objects that you can select and manipulate

Our objects inside the Document Object Model can have properties and methods.
property:describe something about the object.
methods:the thing that the object can do.

假設Object是一輛車
property:車的顏色,座位的數量,車門的數量
mathod:brake(),drive(),park(),其實就是這個物件可以執行的function

We can use javascipt to manipulate our objects ,and it is all done using the dot notation.

get property

to get the value of the property,this is called a getter,if we run the code,it wiil give us a output of red,because the current value of the color property of the car is equal to red.

set property

get property 與 set property的差別於:set property時,we assign a value to it with an equal sign.

call method

以button為例子
