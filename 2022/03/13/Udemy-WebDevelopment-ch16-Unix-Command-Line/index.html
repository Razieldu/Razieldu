Understanding The Command Line . Long Live The Command Line

譬喻為pistachio(開心果)
kennel:果核
shell:外殼

Kernel:refer to the actual program that interfaces with the hardware,it is the core of the operating system

Shell:the user interface for you as a human to be able to interact with the kernel and in turn with the hardware of your computer

有兩種型態的shell,one is graphical user interface shells, the other is command line interface

git bash

bash:Bourne Again Shell,a CLI or a command line interpreter for the UNIX system .

UNIX based system or UNIX like systems(Linux) all over the place.

A lot of servers will run on Unix and even Mac OS 10,run on a unix like operating system

Why Use the command Line

A lot of things that you can do with the command line can be done using the graphical user interface.

It is about greater control and in your journey of becoming a better developer you will need to understand and exert(運用) more power over your tools that is the computer hardware. On a more day to day basis you will find that if you are a regular command line user , that is actually easier and faster to do a lot of really common things,you have full control and flexibility.
