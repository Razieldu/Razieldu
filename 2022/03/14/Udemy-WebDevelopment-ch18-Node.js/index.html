What is Node.js

Why Node.js

我們已經對javascript有初步的認識，Node.js讓我們能使用javascipt程式碼來進行後端的工作,著名的例子:ebay,Uber,Trello,Netflix

Node.js allow us to take Javascript out of the browser and it liberates it ,allowing it to interact directly with a hardware of a computer.

Javascript:give website functionality and behavior,implementing animations,or drop down menus.

Node.js allows us to use Javascript to interact directly with the hardware of the computer, to creat a desktop application

Atom is just based on Node.js.

The power of the command line

建立一個index.js檔,使用Atom開啟

hyper terminal 輸入:node index.js,使用node開啟index.js

the Node REPL(Read Evaluation Print Loops)

It basically allows you to execute code in bite sized chunks,just like that we did with the Chrome Developer Tools Console inside Chrome. We are able to write a single line of code and then hit enter and it would run that line of code and we could run another line of code.

在安裝Node.js時,同時也安裝了NodeREPL

打開Node REPL: write the commandnode and hit enter,and here we can write code directly

離開REPL:.exit 或者是 ctrl + c 兩次

How to use the Native Node Modules(internal Node.js module)

When you install Node ,it already comes bundled with a whole bunch of built in modules. These are essentially libraries of code that the Node team wrote so that they can help you with day to do day things that you might want to do with Node.

We can now use Node.js to interact with the computer directly.

https://nodejs.org/api/ to see all of the native modules and the documentation of how you can use it

打開Atom,建立一個資料夾intro-node,建立一個file1.text,並在檔案內輸入I am file1

到Node.js官網,使用File system module,打開創建的index.js,目的為複製file1

index.js 程式碼

打開hyper terminal,輸入 node index.js,此時確認到檔案1被複製並且檔名為file2.text,如果再執行一次複製檔案的程式碼,則已經存在的file2.text會被新的file2覆蓋

The NPM Package Manager and install External Node Modules

NPM(Node Package Manager):a package manager for external modules,so modules that anybody could build that incorporate into your projects

It actually get bundled with Node,so when we installed Node,we already installed NPM
