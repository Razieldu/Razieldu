FizzBuzzGame

題目

Your program should print each number from 1 to 100 in turn.
When the number is divisible by 3 then instead of printing the number it should print "Fizz".
When the number is divisible by 5, then instead of printing the number it should print "Buzz".
And if the number is divisible by both 3 and 5 e.g. 15 then instead of the number it should print "FizzBuzz".

導師的解法1

個人解法(錯誤版本)

for people in range(1,101):
  if people%3==0:
    print("Fizz")
  elif people%5==0:
    print("Buzz")
  elif people%5==0 and people%3==0:
    print("FizzBuzz")
  else:
    print(people)

思路

  1. 使用%符號找出能被整除的數字
  2. 犯的錯誤
  • 在使用if/else時沒有使用雙等於
  • 設置if/else條件時使用錯誤: if people%3,所有數字都可以是3的被除數,應該設為if people%3==0
  • people%5==0 and people%3==0 這行應該在第一行先執行,因為if/else在條件達成時,就會停止,遇到同時能被5以及3整除的數字會在if people%3==0就停止,顯示Fizz
