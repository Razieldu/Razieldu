React Context Limitations

必須跟一般通用樣式component做區分

  1. Not optimized for high frequency changes
  2. 高頻率改變且必須穿越多個component做使用 => Redux
  3. 短資料傳遞鍊依舊是使用props來進行

Learning the “Rules of Hooks

  1. 只在react的function(1.react component functions 或者 2. custom hooks)呼叫react hooks

  2. only call react hooks at the top level of react component functions

Do not call them in nested functions

Do not call them in any block statements

For useEffect

ALWAYS add everything you refer to inside of the useEffect() as a dependency
