Internal CSS

重點

border為一個矩形,先設置為none,再設置上方為dotted,只顯現上方那條線,border-width調整點的大小,
width調整全條hr的長度

  1. Everything in the web page is a whole bunch of boxes

  2. The border-style property may be specified using one, two, three, or four values.
    When one value is specified, it applies the same style to all four sides.
    When two values are specified, the first style applies to the top and bottom, the second to the left and right.
    When three values are specified, the first style applies to the top, the second to the left and right, the third to the bottom.
    When four values are specified, the styles apply to the top, right, bottom, and left in that order (clockwise).
    Each value is a keyword chosen from the list below.

External CSS

  1. 另外建立一個檔案類別為CSS的檔案,且檔案所在的資料夾必須與index(首頁資料)同個層級,並在個別的網頁都使用link
  2. CSS總檔案頁面中的程式碼
  3. 每一個selector裡面的property依照字母順序依序往下排列,方便尋找

CSS Debug

  1. 使用GOOGLE瀏覽器,點擊右上角三個點,點擊更多工具,點擊開發人員工具,查看裡面產生error的原因
  2. 當三個層級的CSS都同時對一個property做出定義時,優先順序Inline CSS>Internal CSS>External CSS
  3. 總體網頁的風格可以使用external CSS做統一定義,再使用internal CSS以及inline CSS做個別網頁的
