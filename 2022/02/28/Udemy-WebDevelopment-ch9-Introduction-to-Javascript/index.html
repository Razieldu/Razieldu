Java vs Javascript

  1. Javascript: interpret programming language(Ruby,Python)
    Toy language,very slow ,have to execute all the instructions line by line.
    Add some animtion to your website,allow a little bit of user interaction.
    Nowadays javascript can be seen in frameworks ranging from the front end to the back end everything.
    動態程式語言,提供網頁動態互動功能
    It is supported by all major browsers
  2. Java: compiled programming language(C/C++,Swift)
    Serious language, end up with very fast running programs

Java:安卓app
Swift:蘋果app
Javascript:website and web app

naming and naming conventions for javascript variables

  1. give your variables meaningful name
  2. 不能使用與程式既有function一樣的名稱
  3. 開頭不可以是數字
  4. 變數名稱不能包含空格
  5. 變數名稱可以包含字母,數字,dollar sign($),underscore(_)
