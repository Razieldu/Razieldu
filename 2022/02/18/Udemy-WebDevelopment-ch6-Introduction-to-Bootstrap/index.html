What is bootstrap

Front-End-Library

Front-end: the face of the website or mobile app

Back-end: serve or database,basically behind the scenes sort of business logic
how your web site or how your app is going to work,determine how everything is going to work

Front-end language: HTML CSS Javascript
Back-end language:Ruby .Net PHP Java NodeJS SQL

Bootstrapd框架: front-end framework or library

A bunch of code that was written to be reusable that you can drop into your
your site ,and it will make designing the website much,much easier for you

Developed by Twitter in order to try and get consistency across website that they design,free open-source

Responsive:

Users are moving away from desktop towards mobile,a lot of website need to be responsive,means that respond to the view port. 無論你使用什麼工具:手機,平板,或是電腦,都可以提供最佳的瀏覽體驗(best take advantage of the size of the screen real estate) Adaptable-layout Your website changes its design deponds on where it’s being displayed. Bootstrap makes it super easy.

Pre-styled Elements:

有許多現成的模組可以使用,你只需要導入程式碼即可,在製作按鈕的時候,使用HTML按鈕的呈現過於簡單,使用CSS製作,又需要大量程式碼

使用bootstrap製作的按鈕

只是簡單的打出關鍵字,具有margin-radius

https://getbootstrap.com/ (官方網站,有範本能夠使用,以及相關的導入公式)

Install bootstrap

CDN: Content Delivery Network
如果只有一個固定地點可以獲取網站資料,如果距離比較遠,傳輸速度會有所延遲,因此比較有
效率的方法就是在各個主要的地點都有一個地點能夠獲取資料,發出取得資料的請求時,會自動
尋找最接近的地點
In term of bootstrap,they host their bootstrap CSC files on MAX client which has
a whole routing points across the world.

以上解釋了為什麼從bootstrap網站複製導入程式碼的時候,網址是會變動的

  1. 安裝方法1

建置好基本架構之後,到官方網站複製link,在head的部分修改

  1. 安裝方法2 最簡易

直接在官方網站複製starter template ,整個網頁的基本架構

  1. 直接下載網站css資源檔案,在網頁中使用link導入,記得要放在同一個資料夾(bootstrap.css)
    bootstrap的link一定要放在外部css的link之上

使用bootstrap:

完整的頁面左到右用12/12來呈現,分成9/12以及3/12,但是div的最大寬度會隨著頁面縮小跟著縮小,
假設是四張圖片檔也會跟著縮小,不會因為畫面縮小減少呈現的張數(可能是隨著寬度只呈現三張甚至是兩張)

col-md-6:這種欄位的呈現方式會隨著寬度去調整,當寬度不夠呈現6的單位的寬度時,就會維持每個欄位的寬度為
頁面最寬的寬度

程式碼class說明:代表在大頁面的時候進行頁面四等分,中頁面(平板)的時候進行頁面三等分,小頁面(手機)的時候進行頁面兩等分

CSS lINK ORDER

So following that logic, if your custom styles are not overriding the bootstrap styles, all you need to do is move the link to your custom stylesheet to a line after the bootstrap CDN link:

This means that you first load the default bootstrap styles, then you can override some of those components with your own custom CSS.

Unlike CSS and JavaScript, HTML code is executed from top to bottom so the order of your code matters.

Bootstrap container

  1. container
    container 是響應式的, 是一個響應的、固定寬度的 container，內含的 max-width 會在每個斷點中改變數值
    自動responsive,自動更改寬度隨著頁面大小,左右兩邊到頁面邊緣還會有空間

  2. Fluid container

完全填滿頁面寬度,不管今天是哪種頁面寬度
